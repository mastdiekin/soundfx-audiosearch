Мануал по регистрации YandexTranslator API ключа

  1. Переходим по ссылке https://translate.yandex.ru/developers/keys и нажимаем “Создать новый ключ”

  2. Вбиваем любое название и нажимаем “Создать”

  3. Возле нового сгенерированного ключа нажимаем кнопку “Скопировать”

  4. Полученный ключ вставляем в настройках расширения и нажимаем "Сохранить”.

Копия мануала доступна по ссылке
